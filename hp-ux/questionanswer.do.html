hi,

perl depot is compiled with HPUX C-compiler.
You can have a look on the compile options in the file /opt/perl5/lib/5.6.0/PA-RISC1.1/Config.pm.
If you build new modules these compile options are used.
If you don't have HPUX C-Compiler installed,
I think it's better you compile the whole perl with gcc. (I didn't try to change Config.pm.)
